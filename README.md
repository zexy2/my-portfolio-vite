# My Portfolio

Personal portfolio website built with React and Vite. Has a particle animation background using tsParticles.

Live: https://zexy2.github.io/my-portfolio-vite/

## Running locally

```bash
npm install
npm run dev
```

Opens at http://localhost:5173

## Structure

- `src/components/` - React components (Navbar, About, Projects, Contact)
- `src/App.jsx` - Main app with project data
- Deployed to GitHub Pages via Actions

## Editing content

Project data is in `src/App.jsx`. Just update the `projectsData` array to add/edit projects.

## Tech

- React 19
- Vite
- tsParticles
